# Project 2 — Pandas CSV Reader & Basic Analysis

This project explores a Superstore sales dataset using pandas. The goal was to load, inspect, and filter real business data to answer practical questions about sales, profit, customers, and geography.

# What I did
1. Loaded an Excel file into a DataFrame and inspected its structure
  using .head(), .tail(), .info(), and .shape()
2. Generated summary statistics
  with .describe() covering mean, median, spread, and distribution across numeric columns
3. Used .value_counts() to understand order volume across segments, categories, regions, and shipping modes
4. Built a series of filters and aggregations to answer business questions:
- Which category drives the most profit and sales?
- What sub-categories perform best within Technology?
- Which states and cities lead in sales and profit?
- Who are the top 10 most profitable customers?
5. Exported filtered results to CSV
  
# Dataset
Superstore Sales Dataset — available on [Kaggle](kaggle.com)

# Tools used
Python, pandas, Jupyter Notebook
